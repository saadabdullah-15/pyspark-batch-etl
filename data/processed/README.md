# Processed Data

This folder is generated by the PySpark pipeline and is intentionally not committed to Git.

Run the pipeline scripts from the project root to recreate it:

```powershell
python src\01_ingest.py
python src\02_clean.py
python src\03_transform.py
python src\04_validate.py
```

You can also run the complete flow with:

```powershell
python run_pipeline.py
```

Expected generated layers:

```text
data/processed/raw/
data/processed/clean/
data/processed/analytics/
```
